This project is being built as part of my software engineering learning journey to strengthen my skills in:

- JavaScript
- React
- Backend development
- APIs
- Databases
- Authentication
- Full-stack architecture
- Software engineering best practices

The application will continuously evolve as new technologies and concepts are learned and implemented.
